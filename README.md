# 🌐 Trisha Raj — Portfolio Website

This repository contains the source code for my personal **portfolio website**, built using **React + Vite**.  
The website showcases my education, skills, projects, internships, and achievements in a clean, interactive format.

🔗 **Live Website:** https://portfolio-trisha.vercel.app/ 
🔗 **GitHub Profile:** https://github.com/trisharaj11  

---

## ✨ Features

- Responsive and modern UI
- Smooth animations using **Framer Motion**
- Timeline-based sections for Education & Experience
- Projects with GitHub & live demo links
- Clean, recruiter-friendly design
- Fast performance with **Vite**

---

## 🛠 Tech Stack

- **Frontend:** React.js
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Version Control:** Git & GitHub
- **Deployment:** Vercel

---

## 📂 Sections Included

- Hero / Introduction
- Education
